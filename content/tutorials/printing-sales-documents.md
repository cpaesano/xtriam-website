# Printing Sales Documents in bpmPro

When you need to print a proposal or save it as a PDF, bpmPro generates the document directly in your browser. To make sure it looks right, you need to configure two print settings in your browser.

---

## Step 1 — Open the Sales Document Report

Navigate to the sales document you want to print. In the **Sales Document Navigator**, click **View Sales Document** (or use the report toolbar buttons) to open the proposal view.

![Sales document proposal view showing the Customize Report View toolbar and proposal content](/images/tutorials/printing_sales_document_view.jpeg)

---

## Step 2 — Click Print

In the report toolbar at the top, click the **Print** button.

![Sales document with the Print button highlighted in the toolbar](/images/tutorials/printing_click_print.jpeg)

---

## Step 3 — Configure Browser Print Settings

Your browser's print dialog will appear. Before printing or saving as PDF, make sure these two settings are correct under **More settings**:

![Browser print dialog showing the correct settings with Background graphics checked and Headers and footers unchecked](/images/tutorials/printing_browser_options.jpeg)

### Background graphics — Check this box

Make sure **Background graphics** is **checked** (enabled). This ensures the light gray shading, colored headers, and design elements display properly on the printed or PDF version. Without this, the document will look plain and missing visual elements.

### Headers and footers — Uncheck this box

Make sure **Headers and footers** is **unchecked** (disabled). This removes unnecessary information that browsers automatically add — such as the page URL, date, and page numbers — which would clutter your professional proposal.

### Other recommended settings

| Setting | Value |
|---------|-------|
| **Destination** | Your printer, or **Save as PDF** to create a PDF file |
| **Layout** | Portrait |
| **Paper size** | Letter |
| **Margins** | Default |
| **Scale** | Default |

Click **Save** (for PDF) or **Print** to send to your printer.
