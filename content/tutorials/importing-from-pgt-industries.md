# bpmPro — Importing from PGT Industries

> **Purpose:** Export the correct CSV file from the PGT Industries quoting system so it can be imported into bpmPro.
>
> **Before you begin:** Make sure your quote is complete and ready to export in the PGT quoting portal.

---

## Step 1 — Open the Quote Menu and go to Reports

From the Quotes list, locate the quote you want to import. Click the **Menu** button on the right side of the quote row, then select **Reports** from the dropdown.

![Quotes list showing the Menu button with Reports option circled](/images/tutorials/PGT_Choose_Report.png)

---

## Step 2 — Select the "Data Format" report

The **Choose Report** dialog will open. Scroll down to the **Report Type: PGT Report** section and click **Data Format**.

![Choose Report dialog with Data Format option circled](/images/tutorials/PGT_Choose_Report_Data_Format.png)

> **Note:** Do not select *Data Format Detailed*, *Customer Detailed*, or any other report. bpmPro requires **Data Format** specifically.

---

## Step 3 — Download as CSV (comma delimited)

The report will open in a preview window. Click the **export icon** (floppy disk) in the toolbar. A dropdown will appear — select **CSV (comma delimited)**.

![Report preview with export dropdown showing CSV comma delimited selected](/images/tutorials/PGT_Save_Report_as_CSV.png)

> **Note:** Do not select Excel, PDF, or any other format. bpmPro's import feature requires the CSV (comma delimited) format to read the data correctly.

---

## Next Step

Once the CSV file is downloaded, return to bpmPro and use the **Import from PGT** feature to upload the file. bpmPro will automatically parse the line items and populate the order record.

---

## Quick Reference

| Step | Action |
|------|--------|
| 1 | Quotes list → **Menu** button → **Reports** |
| 2 | Choose Report dialog → PGT Report section → **Data Format** |
| 3 | Export icon → **CSV (comma delimited)** → Download |
| 4 | Upload the CSV file into bpmPro |
