Quick Create Sales Document

Overview:
The Quick Create wizard lets you start a new Sales Document in just a few clicks. It creates the Sales Document and optionally a new Project in one step, without navigating through multiple pages.

How to Access:
From the Sales Hub dashboard, click the "Quick Create" card. Or use the "New Sales Document" button available on Account and Project record pages.

How to Use:

From Sales Hub:
1. Click the Quick Create card
2. Search for and select the customer account
3. Select the project (or create a new one)
4. Enter the quote name
5. Click Create
6. The new Sales Document opens ready for adding line items

From Recent Accounts:
1. On the Sales Hub, recent accounts appear as quick-start cards
2. Click one to pre-fill the account
3. Complete the remaining fields and create

From an Account Record:
1. Navigate to the customer's Account page
2. Click "New Sales Document" in the related list
3. Select or create a project
4. The document is created under that account

What Gets Created:
- A new Sales Document with "Unquoted" status
- Linked to the selected Account and Project
- A new Project if you chose to create one
- Ready for adding line items immediately

Tips:
- Use Quick Create from Sales Hub for the fastest workflow
- The recent accounts feature saves time for repeat customers
- After creation, add line items using the Manage Items (A3) section
