Mobile Access

Overview:
Access bpmPro from your mobile device using the Salesforce Mobile App. View records, update projects, and stay connected while in the field.

Getting the App:

iOS (iPhone/iPad):
1. Open App Store
2. Search "Salesforce"
3. Download the official Salesforce app
4. Install and open
5. Log in with your bpmPro credentials

Android:
1. Open Google Play Store
2. Search "Salesforce"
3. Download the official Salesforce app
4. Install and open
5. Log in with your bpmPro credentials

What You Can Do on Mobile:
- View accounts/contacts
- Edit records
- Check project status
- Update opportunities
- View dashboards
- Take photos and attach to records
- Get push notifications

Step-by-Step Guide:

First-Time Setup:
1. Download Salesforce app
2. Open the app
3. Tap "Log In"
4. Enter your username and password
5. Complete any security verification
6. Allow notifications when prompted

Using Offline Mode:
1. Open Settings in the app
2. Go to "Offline"
3. Select records to sync
4. Changes sync automatically when back online

Taking Photos:
1. Open a record (Project, Account, etc.)
2. Tap the camera icon or "Add Photo"
3. Take photo or select from gallery
4. Photo is attached to the record

Common Issues:

"Cannot Log In":
- Verify username and password
- Check internet connection
- Try resetting password

"App Not Syncing":
- Check internet connection
- Force close and reopen app
- Check for app updates

"Missing Features":
- Some features require mobile optimization
- Contact admin if something should be available

Tips:
- Enable push notifications for important alerts
- Download important records before site visits
- Use voice-to-text for quick notes
- Take photos and attach to projects for documentation
- Keep the app updated for best performance
