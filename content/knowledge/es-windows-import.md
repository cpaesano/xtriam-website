ES Windows Integration

Overview:
bpmPro can import quotes directly from ES Windows software, saving you from manual data entry. This feature pulls product specs and pricing from your ES Windows quotes.

How to Access:
- Open an Opportunity or Quote
- Click "Import ES Windows Quote"

Step-by-Step Guide:

Exporting from ES Windows:
1. In ES Windows software, create/open your quote
2. Go to File, then Export or use the export function
3. Choose CSV or Excel format
4. Save the file to your computer

Importing to bpmPro:
1. Open the Opportunity or Quote in bpmPro
2. Click "Import ES Windows Quote" button
3. Click "Choose File" and select your export file
4. Click "Upload"

Reviewing the Import:
1. Preview the imported products in the dialog
2. Verify quantities and specifications
3. Check pricing matches expectations
4. Make any adjustments needed
5. Click "Confirm Import"

What Gets Imported:
- Product Names: Window/door specifications
- Quantities: Number of each item
- Unit Prices: Price per unit
- Total Amounts: Line item totals
- Configurations: Custom specifications

Common Issues:

"Import Failed":
- Check file format (must be CSV or Excel)
- Ensure file isn't corrupted
- Try exporting again from ES Windows
- Check file isn't open in another program

"Products Not Matching":
- Product codes must match your catalog
- Check for typos in ES Windows configuration
- May need to map products manually
- Contact admin for product mapping setup

"Missing Data":
- Some ES Windows fields may not import
- Check if all columns are in the export
- Verify ES Windows export settings

Tips:
- Use consistent product codes between ES Windows and bpmPro
- Always review imports before accepting
- Create a test quote first to verify mapping
- Contact support for mapping issues
