Project Finalization

Overview:
Finalizing a project marks it as complete. The system checks that all required steps are done before allowing finalization, ensuring nothing is missed.

How to Access:
On the Project record page, click the "Finalize" quick action button.

How to Finalize:
1. Open the Project record
2. Click "Finalize"
3. The system runs eligibility checks
4. If any blockers are found (unpaid invoices, incomplete stages), they are listed
5. Resolve any blockers and try again
6. Once all checks pass, confirm the finalization

Eligibility Checks:
The system verifies:
- All production stages are marked as Passed
- Invoices are fully paid (no outstanding balance)
- Required documents are complete

Unfinalizing:
If you need to reopen a finalized project:
1. Open the finalized project
2. Click "Unfinalize"
3. Review the current state
4. The project returns to its previous active stage

Common Issues:

Cannot finalize - blockers shown:
- Review each blocker listed by the system
- Common blockers: unpaid balance, incomplete stages
- Resolve each one and try finalizing again

Tips:
- Review the project financials before finalizing to confirm all costs and payments are recorded
- Finalization is reversible, so do not hesitate to finalize when the project is complete
