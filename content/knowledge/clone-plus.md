Clone Plus - Sales Document Cloning

Overview:
Clone Plus allows you to create an exact copy of a Sales Document, including all its line items (A3 Product & Labor items and C Specialty items). You can clone to the same project or to an entirely different customer and account.

How to Access:
Open any Sales Document record page, then click the "Clone Plus" quick action button in the highlights panel at the top of the page.

Step-by-Step: Clone to Same Project

Use this when you want to create a variation of an existing quote for the same customer, for example with different product options or pricing.

1. Open the Sales Document you want to clone
2. Click the "Clone Plus" button
3. The wizard shows a preview of all line items
4. Select "Same Project" as the destination
5. Enter a name for the new Sales Document
6. Check or uncheck individual line items to include or exclude
7. Click "Clone"
8. The new Sales Document opens automatically with status "Unquoted"

Step-by-Step: Clone to a Different Customer (Re-Bid)

Use this when you want to take an existing quote and clone it for a completely different customer. This is common when you have a standard job scope that applies to multiple clients.

1. Open the Sales Document you want to clone
2. Click the "Clone Plus" button
3. Select "Different Account" as the destination
4. Search for and select the target account (the new customer)
5. Enter the Project Name for the new project that will be created
6. Select the Service Type (Product + Installation, Installation Only, Product Only, Export, or Other)
7. Check or uncheck individual line items to include or exclude
8. Click "Clone"

When cloning to a different account, the system automatically creates a new Project under the target account, a new Sales Document under that project, and copies all selected line items with their pricing, dimensions, and add-ons.

What Gets Cloned:
- All selected A3 Product and Labor line items with dimensions, pricing, and markups
- All A3 Add-Ons (junction records) linked to the cloned items
- All selected C Specialty line items
- The Sales Document name (you can edit it before cloning)

What Does NOT Get Cloned:
- Payment schedules or invoices
- E-signatures
- Attachments or photos
- The original status (always resets to Unquoted)

Common Use Cases:

Re-bidding for a different customer:
- Clone a detailed quote to a new account instead of rebuilding from scratch. This saves significant time when the product specifications are similar.

Creating quote variations:
- Clone a quote within the same project to offer the customer different product options or pricing tiers.

Standardized job scopes:
- Use a template Sales Document and clone it for each new customer who needs the same type of work.

Copying pricing:
- When two customers need the same products, clone the quote and adjust pricing as needed for the new customer.

Common Issues:

Clone button not visible:
- Make sure you are on the Sales Document record page, not the Project page
- The button appears in the quick action bar at the top of the record
- Check with your admin if the action has been added to the page layout

Items missing after clone:
- Only selected (checked) items are cloned. Review the item list before clicking Clone.
- A3 Add-Ons are automatically included when their parent A3 item is selected.

Tips:
- Review the line item list before cloning. You can deselect items you do not need in the new document.
- After cloning, always review the new Sales Document to adjust pricing or details for the new customer.
- The original Sales Document is never modified. Clone Plus only creates copies.
- The new Sales Document always starts as "Unquoted" regardless of the original document's status.
