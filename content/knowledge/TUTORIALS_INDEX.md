## How to Add New Tutorials

1. Read the PDF tutorial from the xTriam Tutorials folder
2. Create a new .md file in `/content/knowledge/`
3. Use the `_TEMPLATE.md` as a starting point
4. Write content in PLAIN TEXT format (no markdown symbols like **, ##, etc.)
5. Add appropriate keywords for AI search matching
6. Update this index file with the new conversion

## Notes

- All knowledge base files use plain text formatting (no markdown symbols)
- Keywords should include common terms users might search for
- Categories: sales, operations, financial, admin, support, general
