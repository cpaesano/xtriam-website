PGT Industries CSV Export

Overview:
This guide shows how to export quotes from the PGT Industries quoting system in CSV format for importing into bpmPro.

Step-by-Step Guide:

Step 1: Go to the Reports Section
- In the PGT quoting system, find the quote you want to export
- Click the "Menu" dropdown button on the quote row
- Select "Reports" from the menu options

Step 2: Choose the Data Format Report
- In the "Choose Report" dialog, look under "Report Type: PGT Report"
- Select "Data Format" from the list

Available report options include:
- AGExport
- Customer Detailed
- Customer Detailed XML
- Customer Summary
- Data Format (select this one)
- Data Format Detailed
- Dealer Cost
- Eagle Data Format Extract
- Line Summary

Step 3: Export as CSV
- Once the report loads, look for the export dropdown
- Click on the export icon
- Select "CSV (comma delimited)" from the options

Export format options available:
- XML file with report data
- CSV (comma delimited) - Select this one
- PDF
- MHTML (web archive)
- Excel
- TIFF file
- Word

Step 4: Save the File
- Save the CSV file to your computer
- Note the location where you saved it

Step 5: Import into bpmPro
- In bpmPro, open the Project or Sales Document
- Click "Import CSV" or "Import Quote"
- Select the CSV file you exported from PGT
- Review and confirm the import

What Gets Imported:
- Product specifications
- Quantities
- Pricing information
- Product descriptions

Common Issues:

"Import Failed":
- Make sure you selected "Data Format" report type
- Ensure the file is saved as CSV format
- Check that the file isn't open in Excel

"Products Not Matching":
- Product codes may need mapping in bpmPro
- Contact your admin to set up product mapping

Tips:
- Always use the "Data Format" report for CSV imports
- Review the imported data before finalizing
- Keep a copy of the original CSV file for reference
