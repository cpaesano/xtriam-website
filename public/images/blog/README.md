## Image Specifications

- **Format**: JPG preferred (smaller file size)
- **Dimensions**: Minimum 1200px wide (will be displayed at 16:9 aspect ratio)
- **File size**: Keep under 300KB per image

## How to Download from Webflow

1. Go to the blog post on www.xtriam.com
2. Right-click the hero/featured image
3. Select "Save image as..."
4. Rename to match the filename above
5. Save to this folder

## After Adding Images

The images will automatically appear on:
- `/blog` - as thumbnails in the grid
- `/post/[slug]` - as hero images at the top of each post
